Microsoft Fabric
Rayfin
AI-Assisted Development (VS Code)
Overview

After many trials, errors, and experiments, I discovered a workflow that dramatically accelerates Microsoft Fabric App development.

The key insight:

A clean semantic model is the foundation of every successful Fabric App.

A well-designed star schema, meaningful measures, clean relationships, and a business-ready semantic layer make the entire application-building process significantly easier.

Architecture
Raw Data
    ↓
Semantic Modeling (Power BI MCP + Altimate AI)
    ↓
Star Schema & Business Logic
    ↓
Microsoft Fabric Semantic Model
    ↓
Rayfin Fabric App
    ↓
Business Application
Workflow
1. Connect Power BI Modeling MCP Server

Connect the Power BI Modeling MCP Server with Altimate AI to enable AI-assisted semantic modeling.

2. Create Semantic Model

Use AI to:

Design the star schema
Create relationships
Generate DAX measures
Build business logic
Optimize the semantic layer
3. Publish to Microsoft Fabric

Deploy the semantic model to a Microsoft Fabric Workspace.

4. Scaffold Rayfin App

Create a new Rayfin application:

npm create @microsoft/rayfin@latest
5. Connect Semantic Model

Connect the Rayfin application back to the semantic model through the Power BI MCP Server.

This allows the application to:

Explore metadata
Understand business entities
Discover measures
Generate contextual user experiences
6. AI-Assisted Development

Use AI tools inside VS Code to:

Generate application components
Build user interfaces
Create business workflows
Refine functionality
7. Deploy

Publish the Fabric App to the Fabric workspace.
